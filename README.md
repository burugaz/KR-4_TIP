# Контрольная работа №4
## Мини-опросник
### https://github.com/burugaz/KR-4_TIP
По итогам работы было сделано веб-приложение, которое позволяет пройти опрос на тему программирования. 
Опрос состоит из 4 вопросов, 3 из которых с выбором ответа, а последний с ручным вводом ответа. 
Есть шкала прогресса ответа на вопросы. Каждый вопрос выводится отдельно. 
В конце опроса пользователю предоставляется возможность просмотреть все выбранные ответы и кнопка для повторного прохождения опроса.
